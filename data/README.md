# Communications with Spammers

## Calls and Messages

Google Voice provides [export service](https://support.google.com/voice/answer/10130510?hl=en) of all the calls and messages via [Google Takeout](https://takeout.google.com/)

## Export all emails from ProntonMail

ProtonMail provides [Proton Mail Export Tool](https://proton.me/support/proton-mail-export-tool) to export all the emails using CLI.
